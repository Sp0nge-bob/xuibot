# Админка

Команда `/admin` (только `BOT_ADMINS`). Навигация — **hub-меню**: корень → раздел → экран.

## Разделы

### Обзор

| Пункт | Назначение |
|-------|------------|
| **Статистика** | Пользователи, подписки, заказы, тикеты, нагрузка VPS (CPU/RAM процессов) |
| **Диагностика** | Техническое состояние: сводка + разделы «Процессы», «Webhook», «VPN», «Хранилище», рекомендации при сбоях |

Диагностика — **не** дубль статистики: только runtime (Telegram, webhook, ноды, Redis, размер `bot.db`), без бизнес-отчётов.

### Монетизация

| Пункт | Назначение |
|-------|------------|
| **Тарифы** | Цены планов (переопределения в БД) |
| **Оплата** | Вкл/выкл способы Platega |
| **Промокоды** | Скидки и бесплатные дни |

### Клиенты

| Пункт | Назначение |
|-------|------------|
| **Клиенты** | Поиск, подписки, удаление |
| **Тикеты** | Поддержка, возвраты, переписка |

### VPN и панели

| Пункт | Назначение |
|-------|------------|
| **Ноды** | Панели 3x-ui, health-check, автосинк, ручная синхронизация |
| **Inbounds** | Какие inbound в подписке |
| **Доступность инбаундов** | Ручное вкл/выкл отображения серверов для пользователей |
| **Лимит IP** | `limitIp` для trial/paid |
| **Happ** | Шифрование ссылки подписки (`none` / `crypt3_local` / `crypt5_api`) |

### Контент

| Пункт | Назначение |
|-------|------------|
| **FAQ** | Статьи с текстом и фото для пользователей |
| **/start** | Приветствие (`{name}`, `{username}`) |
| **Документы** | Политика конфиденциальности и пользовательское соглашение |

### Система

| Пункт | Назначение |
|-------|------------|
| **Бэкап** | Автоархив БД в ЛС, интервал, ручная отправка |
| **Логи** | Выбор файла (`bot.log` + архивы `botlog_*.log`) → последние N строк в `.txt` (100/500/1000/5000 или своё число) |

### Отладка (`ALLOW_DEBUG_ADMIN=true`)

Раздел **Система → Отладка** (с подтверждением входа):

| Пункт | Назначение |
|-------|------------|
| **TEST_MODE** | Вкл/выкл в runtime (БД перекрывает `.env`), сброс к `.env` |
| **Блокировка** | Ручной lockdown: drain PENDING → полная блокировка; whitelist |
| **Пробный** | Лимиты и сброс trial |
| **Сброс данных** | Промокоды, заказы, тикеты, пользователи (опасно) |

Подробнее о lockdown: [Архитектура → Блокировка](architecture.md#блокировка-бота-lockdown).

## Пользовательские функции (не в `/admin`)

| Функция | Где в боте |
|---------|------------|
| Реферальная программа | Главное меню → «Реферальная программа» |
| FAQ | Главное меню → «FAQ» |
| Доступность серверов | Главное меню |
| Документы | Главное меню → «Документы» |
| Поддержка / тикеты | Кнопка «Поддержка» (доступна даже при lockdown) |

Параметры рефералки: [`config/referral.py`](../config/referral.py) (проценты, бонусные дни).
