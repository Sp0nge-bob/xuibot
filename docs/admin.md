# Админка

Команда `/admin` (только `BOT_ADMINS`).

| Раздел | Назначение |
|--------|------------|
| Статистика | Заказы, пользователи, ноды |
| Цены тарифов | Редактирование цен планов |
| Способы оплаты | Вкл/выкл методы Platega |
| Промокоды | Скидки и бесплатные дни |
| Пользователи | Поиск, подписки, удаление |
| Тикеты | Поддержка, возвраты |
| Ноды | Добавление панелей, health-check |
| Inbounds подписки | Какие inbound в выдаче |
| Пробный период | Лимиты trial |
| Экран /start | Приветствие (`{name}`, `{username}`) |
| Отладка | Только при `ALLOW_DEBUG_ADMIN=true` |
